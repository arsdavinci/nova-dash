# シューティングゲーム

## 概要

宇宙を舞台にした縦スクロールシューティングゲーム。スペースシャトルを矢印キーで操作しながら上から来る敵を撃墜し、約30秒でゴールに到達する。HTML5 Canvas + JavaScript 製のブラウザゲーム。

## ゴール

誰でもURLにアクセスするだけでプレイできるWEBゲームアプリとして公開できる状態にする。

## ゲーム仕様

| 項目 | 内容 |
|---|---|
| ジャンル | 縦スクロールシューティング |
| 自機 | スペースシャトル |
| 操作 | 矢印キーで移動、Aキーで弾発射 |
| 背景 | 宇宙背景が上→下にスクロール |
| 敵 | 上から降下、弾が当たると消滅 |
| クリア | 約30秒でゴール出現 |

## 技術スタック

- HTML5 Canvas + JavaScript
- 配信形式：WEBアプリ（ブラウザ完結）
