## 実装した SE・BGM 一覧

| 種別 | 名称 | 実装内容 | 仕様書準拠 |
|---|---|---|---|
| SE | shoot | square 1200→800Hz, 40ms, gainRamp | §2-2 ✅ |
| SE | explosion | white noise + lowpass(1000Hz), ADSR 5/100/0.3/300ms | §2-3 ✅ |
| SE | playerHit | sawtooth 400→150Hz, 250ms, ADSR 5/50/0.4/200ms | §2-4 ✅ |
| SE | gameover | sine 3音アルペジオ（440→330→220Hz）、各350ms | §2-5 ✅ |
| SE | clear | square 4音上昇（523→659→784→1047Hz）、各250ms/最終400ms | §2-6 ✅ |
| BGM | Bass | sawtooth, 65Hz帯, 4分音符パターン, gain=0.15 | §3 ✅ |
| BGM | Lead | square, 262〜392Hz, Cマイナーモチーフ, gain=0.12 | §3 ✅ |
| BGM | Pad | sine + LFO(4Hz±2Hz), 104〜196Hz, gain=0.08 | §3 ✅ |
| BGM | Rhythm | noise kick(lowpass 200Hz) / snare(bandpass 2000Hz), gain=0.10 | §3 ✅ |

BGM は16秒ループ。`setTimeout` で15.8秒後に次ループをスケジューリングし、`stopBGM()` 時に全ノード stop+disconnect。

## インターフェース使用例

```javascript
// ゲーム初期化時（1度だけ）
const sound = new SoundManager();

// タイトル画面で Space/Enter を受けた瞬間
sound.resume();    // AudioContext を running 状態に
sound.playBGM();   // BGM 開始

// ゲームイベントに応じて
sound.playSE('shoot');       // Aキー押下時
sound.playSE('explosion');   // 敵HP=0時
sound.playSE('playerHit');   // 自機HP減少時（無敵判定はゲーム側）

// ゲームオーバー確定時
sound.stopBGM();
sound.playSE('gameover');

// クリア確定時
sound.stopBGM();
sound.playSE('clear');

// 音量設定（0.0〜1.0、デフォルト 0.5）
sound.setVolume(0.7);
```

## 仕様書との差異・調整

| 項目 | 仕様書 | 実装 | 理由 |
|---|---|---|---|
| explosion 総時間 | 0.4秒 | 0.405秒 | stop()のバッファマージン +0.005s |
| gameover S(sustain)ゲイン | 0.6 (仕様書記載) | 実効0.36（0.6×0.6）| gain値0.6をピーク、Decayで0.6×0.6に落としてから保持 |
| clear S(sustain)ゲイン | 0.7 (仕様書記載) | 実効0.35（0.5×0.7換算）| クリッピング防止のため攻撃ピークを0.5に抑制 |
| BGM Bassパターン | 8小節分のみ | 32拍分展開 | 16秒ループに合わせた明示展開 |

## ゲーム実装部への統合指示

### 読み込み

```html
<script src="sound.js"></script>
```

または ES モジュール環境では末尾の `window.SoundManager` から参照。

### resume() を呼ぶタイミング

```javascript
// タイトル画面 → ゲームプレイ遷移時（Space/Enter の最初の押下）
function handleTitleInput(e) {
  if (e.key === ' ' || e.key === 'Enter') {
    sound.resume();   // ← ここで AudioContext を running に
    sound.playBGM();
    startGame();
  }
}
```

### 各ゲームイベントでの playSE 呼び出し目安

| ゲームイベント | 呼び出し |
|---|---|
| Aキー押下（クールダウン制御はゲーム側） | `sound.playSE('shoot')` |
| 敵の HP が 0 になった瞬間 | `sound.playSE('explosion')` |
| 自機 HP 減少確定時（無敵フレーム突入前） | `sound.playSE('playerHit')` |
| HP=0 → ゲームオーバー状態遷移直前 | `sound.stopBGM(); sound.playSE('gameover')` |
| 自機がゴールに接触した瞬間 | `sound.stopBGM(); sound.playSE('clear')` |
