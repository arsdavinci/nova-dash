# Game Design Document — 宇宙シューティング

バージョン: 1.0  
作成: ゲームデザイン部  
対象: ゲーム実装部・ビジュアル部・サウンド部・QA公開部

---

## 1. ゲームコンセプト

宇宙空間をスペースシャトルで駆け抜ける縦スクロールシューティング。プレイヤーは矢印キーでシャトルを操り、Aキーで弾を発射しながら上から迫る UFO や隕石を撃墜する。ゲーム開始から約30秒でゴール（宇宙ステーション）が画面上部から出現し、自機を接触させればクリア。シンプルな操作と短いプレイ時間で、誰でも気軽にブラウザで遊べることを最優先とする。

---

## 2. 画面仕様

| 項目 | 値 |
|---|---|
| Canvas サイズ | 400 × 600 px |
| 背景色 | #000011（深宇宙の濃紺） |
| フレームレート | 60 fps |

### 画面レイアウト図

```
+------------------------------------------+
|  SCORE: 0000        TIME: 30   HP: ●●●  |  ← HUDバー（上端から0px、高さ30px）
+------------------------------------------+
|                                          |
|          [宇宙背景 スクロール中]          |
|                                          |
|          ★ UFO や 隕石が降下             |
|                                          |
|              [自機: シャトル]            |
|                                          |
|                                          |
+------------------------------------------+
  ↑ Canvas全体 400×600px
```

HUD領域（y=0〜29px）はゲームオブジェクトの当たり判定から除外しない（簡略化のため）。

## 3. 状態遷移

```
[タイトル画面]
     ↓ Spaceキー or Enterキー
[ゲームプレイ中]
     ↓ 自機HP=0              ↓ ゴールに接触（30秒後）
[ゲームオーバー]           [クリア画面]
     ↓ Rキー                      ↓ Rキー
[タイトル画面]             [タイトル画面]
```

### 3-1. タイトル画面

| 要素 | 内容 |
|---|---|
| 表示テキスト | ゲームタイトル「SPACE SHUTTLE」（中央上部）、「PRESS SPACE / ENTER」（中央点滅） |
| 背景 | 星スクロールあり |
| 操作 | Space または Enter でゲーム開始 |

### 3-2. ゲームプレイ中

| 要素 | 内容 |
|---|---|
| 表示 | HUD（スコア・残り時間・HP）、自機、弾、敵、背景星 |
| 操作 | 矢印キー移動、Aキー発射 |
| 経過カウント | ゲーム開始から秒単位でカウントアップ（60フレーム=1秒） |

### 3-3. クリア画面

| 要素 | 内容 |
|---|---|
| 表示 | 「MISSION CLEAR!」テキスト、最終スコア、「PRESS R TO RETRY」 |
| 操作 | Rキーでタイトル画面へ |

### 3-4. ゲームオーバー画面

| 要素 | 内容 |
|---|---|
| 表示 | 「GAME OVER」テキスト、最終スコア、「PRESS R TO RETRY」 |
| 操作 | Rキーでタイトル画面へ |

|---|---|
| 幅 | 36 px |
| 高さ | 36 px |
| 初期速度Y | 3 px/frame |
| 速度X | 出現時ランダム -1〜+1 px/frame（直線降下）|

## 7. 背景仕様（星フィールド）

3レイヤー構成。各レイヤーは Canvas 全体を覆い、下端を超えた星は上端に戻る。

| レイヤー | 星の数 | スクロール速度 | 星のサイズ | 備考 |
|---|---|---|---|---|
| 遠景（Far） | 60 個 | 0.5 px/frame | 1 px | 暗めの白 (#888) |
| 中景（Mid） | 30 個 | 1.5 px/frame | 1.5 px | 白 (#ccc) |
| 近景（Near） | 15 個 | 3.0 px/frame | 2 px | 明るい白 (#fff) |

---

## 8. ゴールオブジェクト

| 項目 | 値 |
|---|---|
| 幅 | 64 px |
| 高さ | 64 px |
| 出現タイミング | ゲーム開始30秒後（frame=1800） |
| 出現位置 | x=168（中央揃え）, y=-64（画面外上部） |
| 移動速度Y | 2 px/frame（ゆっくり降下） |
| 当たり判定 | 幅60×高さ60 px（中央揃え） |
| 外観 | 宇宙ステーション（ビジュアル部が実装） |
| クリア判定 | 自機とゴールの当たり判定が重なった瞬間にクリア |

ゴール出現後も時間制限はなし。プレイヤーはゴールまで誘導される。

---

## 9. スコア仕様

| イベント | 加算値 |
|---|---|
| UFO 撃墜 | +100 |
| 隕石 撃墜 | +300 |
| クリアボーナス | +1000 |
| 残HP×ボーナス | 残HP × 200 |

スコアは常時HUDに表示。ゲームオーバー・クリア時に最終スコアを表示。

---

## 10. 当たり判定ルール

- AABB（軸平行矩形）方式
- 弾 vs 敵：弾の矩形と敵の当たり判定矩形が重なれば命中
- 自機 vs 敵：自機の当たり判定と敵の矩形が重なれば被弾
- 自機 vs ゴール：自機の当たり判定とゴールの当たり判定が重なればクリア
- 無敵時間中は自機 vs 敵の判定をスキップ

---

## 11. UI配置詳細

```
y=0〜29px（HUDバー）:
  左端 (x=8):    "SCORE: " + 数値（6桁ゼロ埋め）
  中央 (x=160):  "TIME: " + 残り秒数（2桁）
  右端 (x=320):  "HP: " + ●×HP数（●=残、○=消費）

フォント: 14px monospace（白文字）
```

## 12. 実装優先度

| 優先度 | 機能 |
|---|---|
| 必須 | タイトル・ゲームプレイ・クリア・ゲームオーバー状態遷移 |
| 必須 | 自機移動・弾発射・当たり判定 |
| 必須 | UFO・隕石の出現・撃墜 |
| 必須 | 星背景スクロール |
| 必須 | HUD（スコア・時間・HP）|
| 必須 | ゴール出現・クリア判定 |
| 推奨 | 無敵時間点滅 |
| 推奨 | 難易度変化（時間経過で速度増加）|
| オプション | 隕石の残HP色変化 |
