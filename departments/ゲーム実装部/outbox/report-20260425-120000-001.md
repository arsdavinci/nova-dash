## 修正サマリー

### 座標変換（左上座標 → 中心座標）

| ファイル | 変換内容 |
|---|---|
| player.js | `cx = x + width/2, cy = y + height/2` として `ShipVisual.draw(ctx, cx, cy, width, height, frame)` に渡す |
| enemy.js | 同上。`EnemyVisual.draw(ctx, type, cx, cy, width, height, frameCount)` |
| bullet.js | `cx = x + width/2, cy = y + height/2` として `ObjectsVisual.drawBullet(ctx, cx, cy)` |
| game.js (Goal) | `cx = x + width/2, cy = y + height/2` として `ObjectsVisual.drawGoal(ctx, cx, cy, frame)` |

### サウンド統合のポイント

- `sound = new SoundManager()` を DOMContentLoaded で生成
- `keydown` / `click` の初回操作で `sound.resume()` を呼ぶ（ブラウザ自動再生ポリシー対応）
- タイトル→プレイ遷移の `initGame()` 内で `sound.playBGM()`
- クリア/ゲームオーバー遷移で `sound.stopBGM()` + `sound.playSE('clear'|'gameover')`
- 弾発射: `sound.playSE('shoot')`
- 敵撃墜: `sound.playSE('explosion')`
- 自機被弾: `sound.playSE('playerHit')`
- 旧コードの `Sound.playSE('hit')` を `sound.playSE('playerHit')` に修正（IF不一致を解消）

### BackgroundVisual 統合

- `background.js` を `BackgroundVisual` への薄いラッパーに全面書き換え
- `BackgroundVisual.init` / `.update` / `.draw` を使用、既存の Background クラス IF はそのまま維持

### 爆発エフェクト統合

- `enemy.js` に `exploding`, `explosionFrame`, `explosionCX/CY` フィールド追加
- 撃墜時（`takeDamage()` で hp=0）に爆発フラグを立て、中心座標を記録
- `updateExplosion()` メソッド追加し game.js の更新ループで呼ぶ
- `draw()` 内で `EffectsVisual.drawExplosion(ctx, cx, cy, frame, 20)` を呼ぶ
- enemies 配列のフィルタを `active || exploding` に変更（爆発終了まで描画継続）

### script 読み込み順序（index.html）

```
sound.js
→ background_visual.js → ship_visual.js → enemy_visual.js
→ objects_visual.js → effects_visual.js
→ collision.js → background.js → bullet.js → enemy.js → player.js → game.js
```
