## バッチ2（ビジュアル部・サウンド部統合）での注意点

### ビジュアルモジュールのフックポイント

| クラス | ガード変数 | 期待する呼び出し |
|---|---|---|
| `player.draw()` | `typeof ShipVisual` | `ShipVisual.draw(ctx, x, y, invincible > 0)` |
| `enemy.draw()` UFO | `typeof EnemyVisual && EnemyVisual.drawUFO` | `EnemyVisual.drawUFO(ctx, x, y, hp)` |
| `enemy.draw()` 隕石 | `typeof EnemyVisual && EnemyVisual.drawMeteor` | `EnemyVisual.drawMeteor(ctx, x, y, hp)` |
| `bullet.draw()` | `typeof BulletVisual` | `BulletVisual.draw(ctx, x, y)` |
| `Goal.draw()` | `typeof GoalVisual` | `GoalVisual.draw(ctx, x, y)` |
| `Background.draw()` | `typeof BackgroundVisual` | `BackgroundVisual.draw(ctx, stars, W, H)` |

### 座標系の注意

- `player.js` / `enemy.js` / `bullet.js` いずれも `x, y` は **左上座標**
- ビジュアルモジュールが中心座標を要求する場合、統合時に `cx = x + width/2, cy = y + height/2` で変換が必要

### サウンドのフックポイント

| タイミング | 呼び出し |
|---|---|
| PLAYING 開始 | `Sound.init()` / `Sound.playBGM()` |
| 弾発射 | `Sound.playSE('shoot')` |
| 敵撃墜 | `Sound.playSE('explosion')` |
| 自機被弾 | `Sound.playSE('hit')` ← フォールバック版。統合版では `playerHit` に変更必要 |
| CLEAR | `Sound.stopBGM()` + `Sound.playSE('clear')` |
| GAMEOVER | `Sound.stopBGM()` |

> **重要**: フォールバック版（outbox/player.js）は `Sound.playSE('hit')` を呼ぶが、SoundManager のメソッド名は `playSE('playerHit')`。統合版では修正が必要。
