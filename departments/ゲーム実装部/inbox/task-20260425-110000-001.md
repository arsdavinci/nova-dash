## フォールバック描画の扱い

バッチ2でビジュアル部の描画モジュールを統合するが、**今回はビジュアルモジュール無しで動作する**ことを必須とする。

各 draw メソッドは：

```javascript
draw(ctx) {
  if (typeof ShipVisual !== "undefined") {
    ShipVisual.draw(ctx, this.x, this.y, this.invincible > 0);
  } else {
    // フォールバック矩形描画
    ctx.fillStyle = ...;
    ctx.fillRect(...);
  }
}
```

の形式で、**今回はフォールバック側を確実に実装**。ShipVisual 等の参照は `typeof` ガードで安全に。

同様にサウンドも `typeof Sound !== "undefined"` で分岐し、無くても動く設計にする。
