# シューティングゲーム — プロジェクトコンテキスト

このファイルは `projects/shooting-game/` 配下で Claude Code が起動された時にロードされる。

## プロジェクト概要

宇宙を舞台にした縦スクロールシューティングゲーム。スペースシャトルを矢印キーで操作し、Aキーで弾を発射。上から来る敵を撃墜しながら約30秒でゴールに到達する。

## ゴール

HTML5 Canvas + JavaScript で実装した縦スクロールシューティングゲームを完成させ、誰でもアクセスできるWEBゲームアプリとして公開できる状態にする。

## タイプ

game（WEBゲームアプリ）

## 技術スタック

- HTML5 Canvas + JavaScript
- 外部ライブラリ：未定（必要に応じて選定）
- 配信形式：静的WEBアプリ（ブラウザ完結）

## ゲーム仕様

- 自機：スペースシャトル、矢印キーで上下左右移動
- 攻撃：Aキーで前方に弾を発射
- 敵：上から降下、弾が命中すると消滅
- 背景：宇宙空間、上→下スクロール
- クリア条件：約30秒後にゴールが出現
